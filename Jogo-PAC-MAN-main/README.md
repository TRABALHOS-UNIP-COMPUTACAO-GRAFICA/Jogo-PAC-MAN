🎮 Pac-Man em Computação Gráfica

Este projeto apresenta uma réplica do clássico jogo Pac-Man, desenvolvida como parte dos estudos na disciplina de Computação Gráfica.

O objetivo principal foi aplicar, na prática, os conceitos aprendidos em aula, com foco especial na utilização de primitivas gráficas para a construção dos elementos visuais do jogo.

📚 Conceitos Aplicados

Durante o desenvolvimento, foram explorados os seguintes fundamentos:
	•	Primitivas gráficas (pontos, linhas, retângulos, círculos)
	•	Sistema de coordenadas
	•	Manipulação de cores
	•	Atualização de tela (renderização)
	•	Movimentação e colisão de objetos
	•	Estrutura básica de um game loop

🕹️ Sobre o Jogo

O projeto recria os principais elementos do jogo original:
	•	Personagem principal (Pac-Man)
	•	Labirinto
	•	Pontos coletáveis
	•	Sistema de movimentação
	•	Interação com inimigos (quando implementado)

🎯 Objetivo Acadêmico

Mais do que apenas reproduzir um jogo clássico, este projeto tem como finalidade consolidar os conhecimentos em:
	•	Estruturas gráficas
	•	Lógica de programação
	•	Organização de código
	•	Desenvolvimento de aplicações interativas

🚀 Tecnologias Utilizadas
	•	Python
	•	Pygame
